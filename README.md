# 📊 Seaborn for Data Science

A comprehensive collection of Seaborn visualizations, practice exercises, and mini-projects for Data Science and Data Analysis.

## 🚀 About

This repository contains my journey of learning and implementing Seaborn, a powerful Python data visualization library built on top of Matplotlib.

The repository includes:

- Seaborn fundamentals
- Statistical visualizations
- Categorical plots
- Distribution plots
- Correlation analysis
- Exploratory Data Analysis (EDA)
- Dashboard-style projects
